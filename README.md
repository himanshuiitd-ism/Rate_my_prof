# rate_my_prof

This repo contains a simple MVP for a site that scrapes faculty from `https://www.iitism.ac.in/all-faculty` and allows anonymous ratings. A lightweight message feature stores comments in the database via regular REST calls.

Folders:

- `backend/` — Express + MongoDB (or in-memory for dev)
- `frontend/` — React (Vite)

See `backend/README.md` for backend-specific notes.
